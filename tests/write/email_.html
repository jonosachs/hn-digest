1. I built Timeframe, our family e-paper dashboard

URL: https://hawksley.org/2026/02/17/timeframe.html

Confidence: high

Summary: This article explores the decade-long development of 'Timeframe,' a custom ambient home dashboard. The author details the transition from traditional backlit LCD 'Magic Mirrors' to electrophoretic (e-paper) displays, which reflect ambient light rather than emitting it, making them less visually intrusive at night. The technical progression moves from jailbroken Kindles to professional Visionect hardware and finally to high-resolution Boox Mira Pro monitors capable of real-time updates via HDMI.

Significance: It demonstrates how to iterate on a long-term side project by balancing hardware constraints, such as refresh rates and power consumption, with a simplified software backend.

Key Terms:

  • E-paper (Electrophoretic Display) - A display technology that mimics the appearance of ordinary ink on paper by using electrically charged pigment particles.
  • Long-polling - A web communication pattern where the client requests data from the server and the server holds the request open until new information is available.
  • Home Assistant - An open-source home automation platform that acts as a central hub for controlling and monitoring smart devices.
  • Headless Browser (IMGKit) - A web browser without a graphical user interface, used here to render HTML/CSS into static image files (PNGs) for low-power displays.

2. Show HN: WARN Firehose – Every US layoff notice in one searchable database

URL: https://warnfirehose.com

Confidence: high

Summary: WARN Firehose is a data engineering project that aggregates layoff notices filed under the U.S. Worker Adjustment and Retraining Notification (WARN) Act. Because these notices are published by 50 different state agencies in fragmented formats—including PDFs, Excel files, and HTML tables—the project focuses on the 'Extract, Transform, Load' (ETL) process to create a unified, searchable database. The platform provides access through a REST API and high-performance data formats like Parquet for analytical processing.

Significance: It illustrates a practical application of data scraping and normalization to solve the 'siloed data' problem, turning fragmented public records into a high-value structured dataset.

Key Terms:

  • WARN Act - A U.S. labor law requiring employers with 100 or more employees to provide 60 days' notice of plant closings or mass layoffs.
  • Data Normalization - The process of organizing data from disparate sources into a consistent, standardized format to reduce redundancy and improve integrity.
  • Parquet - An open-source, column-oriented data file format designed for efficient data storage and retrieval in analytical workloads.
  • JSON-LD - A method of encoding linked data using JSON, designed to make data machine-readable in a way that maps to the Semantic Web.
